### MarkDown 插入音频

```markdown
<iframe frameborder="no" border="0" marginwidth="0" marginheight="0" width=330 height=86 src="音频地址"></iframe>
```

### MarkDown 插入视频

```markdown
<iframe height=498 width=510 src="视频地址" frameborder=0 allowfullscreen></iframe>
```
